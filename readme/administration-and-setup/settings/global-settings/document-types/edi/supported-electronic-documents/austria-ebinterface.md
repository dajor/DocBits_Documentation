# 🇦🇹 AUSTRIA EBINTERFACE

| Property | Value |
|----------|-------|
| **Country / Region** | Austria |
| **Document Types** | Invoice, Credit Note |
| **Format** | XML |
| **Standard** | ebInterface |

ebInterface is the Austrian e-invoicing standard maintained by the Austrian Federal Economic Chamber (WKÖ). It defines a structured XML format for electronic invoices used primarily in B2G (business-to-government) transactions in Austria. DocBits supports versions 4.3 through 6.1.
