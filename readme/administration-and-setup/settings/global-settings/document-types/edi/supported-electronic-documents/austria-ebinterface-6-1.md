# 🇦🇹 AUSTRIA EBINTERFACE 6.1

| Property | Value |
|----------|-------|
| **Country / Region** | Austria |
| **Document Types** | Invoice, Credit Note |
| **Format** | XML |
| **Standard** | ebInterface 6.1 |

ebInterface 6.1 is the latest version of the Austrian e-invoicing standard. It includes updated validation rules, improved support for credit notes, and enhanced compatibility with the Peppol network for cross-border invoicing.
