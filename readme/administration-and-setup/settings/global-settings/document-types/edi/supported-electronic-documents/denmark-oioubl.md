# 🇩🇰 DENMARK OIOUBL

| Property | Value |
|----------|-------|
| **Country / Region** | Denmark |
| **Document Types** | Invoice, Credit Note |
| **Format** | UBL XML |
| **Standard** | OIOUBL |

OIOUBL (Offentlig Information Online - Universal Business Language) is the Danish national e-invoicing standard based on UBL 2.0. It is used for electronic invoicing in both B2G and B2B contexts in Denmark. DocBits supports OIOUBL 2.1.
