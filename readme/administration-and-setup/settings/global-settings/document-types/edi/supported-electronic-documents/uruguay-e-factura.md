# 🇺🇾 URUGUAY E-FACTURA

| Property | Value |
|----------|-------|
| **Country / Region** | Uruguay |
| **Document Types** | e-Invoice (e-Factura) |
| **Format** | XML |
| **Standard** | e-Factura (CFE Type 111) |

Uruguay e-Factura is the electronic invoice document type within the CFE system, identified by type code 111. It is used for formal B2B transactions between RUT-registered entities and includes all tax identification and computation fields required by the DGI.
