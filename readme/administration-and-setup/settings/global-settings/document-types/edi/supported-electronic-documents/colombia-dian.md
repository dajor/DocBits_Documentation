# 🇨🇴 COLOMBIA DIAN

| Property | Value |
|----------|-------|
| **Country / Region** | Colombia |
| **Document Types** | Invoice (Factura Electrónica), Credit Note (Nota Crédito), Support Document (Documento Soporte) |
| **Format** | UBL 2.1 XML |
| **Standard** | DIAN (Dirección de Impuestos y Aduanas Nacionales) |

Colombia DIAN is the Colombian electronic invoicing system operated by the national tax authority. It uses UBL 2.1 as its base format with Colombian extensions for tax identification (NIT), withholding taxes (retenciones), and the CUFE/CUDE unique invoice identifiers. DocBits supports both Factura Electrónica and Documento Soporte document types.
