# 🇨🇱 CHILE DTE

| Property | Value |
|----------|-------|
| **Country / Region** | Chile |
| **Document Types** | Invoice, Credit Note, Debit Note, Dispatch Guide |
| **Format** | XML |
| **Standard** | DTE (Documento Tributario Electrónico) |

DTE (Documento Tributario Electrónico) is the Chilean electronic tax document standard regulated by the Servicio de Impuestos Internos (SII). It covers multiple document types identified by type codes (33=Factura, 52=Guía de Despacho, 61=Nota de Crédito). DocBits auto-detects the DTE type code and extracts all relevant fields.
