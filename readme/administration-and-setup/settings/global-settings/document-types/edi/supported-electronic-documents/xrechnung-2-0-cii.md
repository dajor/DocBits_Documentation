# 🇩🇪 XRECHNUNG 2.0 CII

| Property | Value |
|----------|-------|
| **Country / Region** | Germany |
| **Document Types** | Invoice, Credit Note |
| **Format** | CII |
| **Standard** | XRechnung 2.0 |
| **Based On** | EN 16931 (UN/CEFACT Cross Industry Invoice) |

XRechnung 2.0 implements the EN 16931 Core Invoice standard with German CIUS (Core Invoice Usage Specification) extensions. This version introduced a major restructuring of the standard with updated validation rules and improved compliance with the European Norm.
