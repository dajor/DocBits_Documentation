# 🇸🇬 INVOICENOW

| Property | Value |
|----------|-------|
| **Country / Region** | Singapore |
| **Document Types** | Invoice, Credit Note |
| **Format** | UBL 2.1 XML (Peppol) |
| **Standard** | InvoiceNow (IMDA) |

InvoiceNow is Singapore's national e-invoicing initiative built on the Peppol network, led by IMDA (Infocomm Media Development Authority). It enables businesses to exchange structured electronic invoices through Peppol Access Points using the UBL 2.1 format.
