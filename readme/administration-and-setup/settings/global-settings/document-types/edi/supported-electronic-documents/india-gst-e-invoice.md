# 🇮🇳 INDIA GST E-INVOICE

| Property | Value |
|----------|-------|
| **Country / Region** | India |
| **Document Types** | Invoice, Credit Note, Debit Note |
| **Format** | JSON |
| **Standard** | GST E-Invoice (GSTN e-Invoice System) |

India GST E-Invoice is the electronic invoice standard under India's Goods and Services Tax regime, operated by GSTN (GST Network). Businesses above the turnover threshold must generate e-Invoices through the Invoice Registration Portal (IRP), which returns a signed JSON with an IRN (Invoice Reference Number) and QR code.
