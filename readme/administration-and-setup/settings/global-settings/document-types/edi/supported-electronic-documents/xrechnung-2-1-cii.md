# 🇩🇪 XRECHNUNG 2.1 CII

| Property | Value |
|----------|-------|
| **Country / Region** | Germany |
| **Document Types** | Invoice, Credit Note |
| **Format** | CII |
| **Standard** | XRechnung 2.1 |
| **Based On** | EN 16931 (UN/CEFACT Cross Industry Invoice) |

XRechnung 2.1 implements the EN 16931 Core Invoice standard with German CIUS (Core Invoice Usage Specification) extensions. This version introduced improved validation with enhanced Schematron rules and stricter conformance checks.
