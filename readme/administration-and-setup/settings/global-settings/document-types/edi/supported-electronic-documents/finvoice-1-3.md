# 🇫🇮 FINVOICE 1.3

| Property | Value |
|----------|-------|
| **Country / Region** | Finland |
| **Document Types** | Invoice |
| **Format** | XML (SOAP envelope) |
| **Standard** | Finvoice 1.3 |

Finvoice 1.3 is an earlier version of the Finnish e-invoicing standard maintained by Finance Finland. It uses a SOAP message envelope and defines a structured XML format for B2B invoicing between Finnish banks and their corporate clients.
