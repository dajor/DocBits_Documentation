# 🇯🇵 JP PINT SELF-BILLING

| Property | Value |
|----------|-------|
| **Country / Region** | Japan |
| **Document Types** | Self-Billed Invoice |
| **Format** | UBL 2.1 XML |
| **Standard** | JP PINT Self-Billing |

JP PINT Self-Billing is the self-billing variant of the Japanese Peppol International invoice model. It enables buyers to issue invoices on behalf of suppliers, following the same JP PINT structure with reversed party roles, as permitted under specific conditions in Japanese tax law.
