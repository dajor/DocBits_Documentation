# 🇦🇹 AUSTRIA EBINTERFACE 6.0

| Property | Value |
|----------|-------|
| **Country / Region** | Austria |
| **Document Types** | Invoice, Credit Note |
| **Format** | XML |
| **Standard** | ebInterface 6.0 |

ebInterface 6.0 introduced alignment with the European standard EN 16931 while maintaining backward compatibility with Austrian-specific requirements. It supports the structured representation of invoice data including line items, tax details, and payment information.
