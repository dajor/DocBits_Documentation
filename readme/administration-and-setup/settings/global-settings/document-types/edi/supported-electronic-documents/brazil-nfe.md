# 🇧🇷 BRAZIL NFE

| Property | Value |
|----------|-------|
| **Country / Region** | Brazil |
| **Document Types** | Invoice (Nota Fiscal Eletrônica) |
| **Format** | XML |
| **Standard** | NF-e (Nota Fiscal Eletrônica) |

NF-e (Nota Fiscal Eletrônica) is the Brazilian electronic invoice for goods and interstate commerce, regulated by SEFAZ. Each NF-e document contains detailed product information, tax calculations (ICMS, IPI, PIS, COFINS), and a unique access key (chave de acesso) for validation.
