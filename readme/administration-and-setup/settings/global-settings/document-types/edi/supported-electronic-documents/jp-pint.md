# 🇯🇵 JP PINT

| Property | Value |
|----------|-------|
| **Country / Region** | Japan |
| **Document Types** | Invoice |
| **Format** | UBL 2.1 XML |
| **Standard** | JP PINT (Peppol International Model for Japan) |

JP PINT is the Japanese implementation of the Peppol International (PINT) invoice model. It adapts the global Peppol standard to Japanese business requirements, including support for the Qualified Invoice System (適格請求書) introduced under the revised Consumption Tax Act.
