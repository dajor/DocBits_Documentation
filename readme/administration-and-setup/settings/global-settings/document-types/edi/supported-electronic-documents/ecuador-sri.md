# 🇪🇨 ECUADOR SRI

| Property | Value |
|----------|-------|
| **Country / Region** | Ecuador |
| **Document Types** | Invoice, Credit Note, Withholding Certificate, Dispatch Guide |
| **Format** | XML |
| **Standard** | SRI (Servicio de Rentas Internas) |

The Ecuadorian SRI electronic invoicing system is operated by the national tax authority (Servicio de Rentas Internas). Electronic documents must be authorized by the SRI before being delivered to recipients. DocBits supports multiple SRI schema versions from 1.0.0 to 2.1.0.
